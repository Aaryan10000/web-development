# 😂 Make Me Laugh – Joke Fetching Web App

A simple and fun web app built to bring a smile to your face!  
It fetches random jokes and displays them instantly — try not to laugh 😄
---

##  Features

- **Random Joke Generator:** Fetches a new joke every time you click the button.  
- **Recent Jokes List:** Keeps track of the jokes you've already seen.  
- **Responsive Design:** Works smoothly on both desktop and mobile screens.  
- **Minimal & Clean UI:** Simple layout built using HTML, CSS, and JavaScript.  
- **Fun Experience:** Built to make you laugh anytime, anywhere!

---

## Tech Stack

- **HTML5** – Structure of the app  
- **CSS3** – Styling and layout  
- **JavaScript (Vanilla JS)** – Logic for fetching and displaying jokes  

---

##  How to Use

1. Open the app in your browser.  
2. Click on **"Next Joke"** to fetch a random joke.  
3. Laugh out loud 😆  
4. Check out the list of **recent jokes** below!
